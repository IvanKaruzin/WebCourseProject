# Архитектура веб-приложения «CineVault»

## Назначение системы

Веб-приложение представляет собой онлайн-кинобиблиотеку, позволяющую пользователям просматривать информацию о фильмах, искать фильмы по различным критериям, оставлять отзывы и формировать собственную коллекцию избранных фильмов.

Система реализована по архитектуре Client–Server с разделением FrontEnd и BackEnd частей.

# Страницы приложения

```text

```

* отображение популярных фильмов;

## Каталог фильмов

# Импорт данных

При первом запуске система импортирует данные из датасета IMDb Top 1000 Movies and TV Shows.

Во время импорта:

* создаются фильмы;
* создаются жанры;
* создаются режиссёры;
* создаются актёры;
* формируются связи между объектами.

---

# Тестирование

Для серверной части реализуются Unit-тесты.

Проверяются:

* регистрация пользователя;
* авторизация пользователя;
* получение списка фильмов;
* получение информации о фильме;
* создание отзыва;
* добавление фильма в избранное.

---

# Дополнительные возможности

Для получения дополнительных баллов реализуются следующие функции:

# Пользовательский сценарий работы

1. Пользователь открывает главную страницу.
2. Переходит в каталог фильмов.
3. Выполняет поиск интересующего фильма.
4. Открывает страницу фильма.
5. Регистрируется или входит в систему.
6. Добавляет фильм в избранное.
7. Оставляет отзыв и оценку.
8. Просматривает список избранных фильмов в личном кабинете.
